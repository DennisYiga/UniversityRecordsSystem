# University Records Management System

A Java-based console application for managing academic data. This project demonstrates core Object-Oriented Programming (OOP) principles and effective use of the Java Collections Framework.

## 🛠️ Technical Features
- **Student & Course Management**: Uses `HashMap` for efficient data retrieval.
- **Encapsulation**: Strict use of private fields and public accessors to protect data integrity.
- **Static Members**: Implements global tracking for total enrollments across all courses.
- **GPA Calculation**: Modular logic to calculate averages based on enrolled course grades.

## 📁 Project Structure
- `src/`: Contains the Java source code.
- `.gitignore`: Configured to exclude compiled `.class` files.
